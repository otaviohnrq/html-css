Para marcar um texto, é utilizado a tag MARK, como nesse trecho.

Para desconsiderar um texto, é utilizado a tag DEL, o vitória é o maior da Bahia.

Já pelo contrário, para ter um texto inserido é utilizado a tag INS, Ana Beatriz casa comigo?

As tag SUB e SUP servem para escrever H2O e X2, assim: H2O e X2, detalhe: a tag é utilizada no caractere que deseja ficar no alto ou embaixo
